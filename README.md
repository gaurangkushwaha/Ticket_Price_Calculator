# Ticket Price Calculator in Python

## 📌 Overview
This Python project calculates ticket prices based on:
- Age
- Student status

The program uses conditional statements (`if-elif-else`) and nested conditions to apply different pricing rules.

## 🚀 Features
- Age-based ticket pricing
- Student discount system
- Nested conditional statements
- User input handling
- Beginner-friendly Python logic

---

## 🛠️ Technologies Used
- Python 3
